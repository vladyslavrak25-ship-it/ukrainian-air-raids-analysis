# Статистичний аналіз та декомпозиція часових рядів повітряних тривог в Україні

Репозиторій містить аналітичний проєкт із дослідження історичних даних повітряних тривог в Україні. Метою роботи є виявлення прихованих закономірностей, часової циклічності та аномалій у сигналах цивільного захисту за допомогою методів Data Science.

## 📊 Основні статистичні показники (Загальний масив: 273 274 записи)

На основі первинного аналізу тривалості тривог (`duration_minutes`) було отримано такі ключові метрики:
* **Мінімум:** 0.0 хв (короткострокові превентивні сигнали або технічні збої системи).
* **25% (Перший квартиль):** 27.2 хв.
* **50% (Медіана):** 48.55 хв — половина всіх повітряних тривог в Україні завершується в межах 49 хвилин.
* **75% (Третій квартиль):** 119.78 хв — близько 2 годин (що математично збігається з тактико-технічними лімітами перебування в повітрі основних носіїв аеробалістичного озброєння РФ).
* **Максимум:** 869 836.46 хв (~604 дні). Це довготривала аномалія, спричинена перманентним статусом повітряної тривоги в окупованих та прикордонних регіонах (зокрема, безперервна тривога у Луганській області).

## ⚙️ Обробка екстремальних значень (Outliers)

Через наявність тривог, що тривають місяцями, середнє арифметичне значення (`mean = 134.75 хв`) виявилося сильно зміщеним і не відображало реальну картину для більшості областей. 

**Стратегія очищення даних:**
За рекомендацією комісії, для забезпечення математичної коректності моделей та запобігання викривленню результатів, усі екстремальні випадки тривалістю понад 24 години були **повністю вилучені з основного датафрейму (`df`)** перед проведенням фінального аналізу. Це дозволило очистити часовий ряд від екстремальних викидів, хоча й призвело до втрати безперервного регіонального сигналу по окремих прифронтових зонах.

## 📈 Методологія дослідження

1. **Експлоративний аналіз даних (EDA):** Розрахунок дескриптивних статистик, виявлення та ізоляція аномалій.
2. **Аналіз часових патернів:** Дослідження щільності запусків за годинами доби (виявлено стійкі піки у вечірній та нічний час) та днями тижня (статистично підтверджено рівномірний розподіл).
3. **Математична декомпозиція:** Застосування адитивної моделі часових рядів (`seasonal_decompose`) для розкладання сигналу на три складові:
   * **Тренд (Trend):** відображає довгострокові макрозміни та інтенсивність воєнних кампаній.
   * **Сезонність (Seasonal):** виділення стійкого 7-денного календарного циклу.
   * **Залишок (Residual):** випадкові високоамплітудні сплески, які чітко корелюють із датами масштабних комбінованих ракетно-дронових обстрілів.
